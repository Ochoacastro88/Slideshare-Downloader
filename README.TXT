# Slideshare Downloader

## Description
A simple web app to download images from a given URL and compile them into a PDF.

## Features
- Search for images from a URL.
- Select images for download.
- Create a PDF with selected images.

## Installation
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`

## Usage
Visit the app in your browser at `http://127.0.0.1:5000/`.

# Descargador de Slideshare

## Descripción
Una aplicación web sencilla para descargar imágenes desde una URL proporcionada y compilarlas en un archivo PDF.

## Características
- Buscar imágenes desde una URL.
- Seleccionar imágenes para descarga.
- Crear un PDF con las imágenes seleccionadas.

## Instalación
1. Clona este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la app: `python app.py`

## Uso
Visita la app en tu navegador en `http://127.0.0.1:5000/`.
